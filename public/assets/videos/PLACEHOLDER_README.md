# Video Assets Placeholder

## Required Videos:

### Hero Background Video
- `hero-background.mp4` - Looping background video for hero section
  - Recommended specs:
    - Duration: 10-20 seconds (looping)
    - Resolution: 1920x1080 (Full HD minimum)
    - Format: MP4 (H.264 codec)
    - File size: < 5MB (optimize for web)
    - Frame rate: 30fps
    - Audio: None (remove audio track for smaller file size)

### Optional Demo Videos
- `product-demo.mp4` - Product walkthrough/demo video
- `testimonial-video.mp4` - Customer testimonial video

## Video Optimization Tips:
1. Use tools like HandBrake or FFmpeg to compress videos
2. Consider using a CDN for video hosting (Cloudflare, AWS CloudFront)
3. Implement lazy loading for videos below the fold
4. Provide poster images for videos
5. Consider using HTML5 video with fallbacks

## Alternative Solutions:
- Use animated gradients or CSS animations instead of video backgrounds
- Host videos on YouTube/Vimeo and embed them
- Use lightweight animated SVGs or Lottie animations
